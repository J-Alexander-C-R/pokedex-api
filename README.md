## Descripción del proyecto

Este proyecto consiste en una aplicación web desarrollada con HTML, CSS y JavaScript que consume dos APIs públicas utilizando fetch y async/await.

La aplicación funciona como una Pokédex básica, permitiendo buscar Pokémon por nombre y visualizar información de este, como su tipo o peso. Además, incluye una sección independiente que muestra chistes aleatorios obtenidos desde otra Api, demostrando el consumo de múltiples Apis externos en una misma aplicación.


## APIs utilizadas

### PokéAPI

* Uso: Obtener información de Pokémon.
* Datos mostrados:

  * Nombre
  * Imagen
  * Altura
  * Peso
  * Tipo(s)

### JokeAPI

* Uso: Obtener chistes aleatorios.
* Funcionalidad: Mostrar un chiste nuevo cada vez que el usuario presiona el botón correspondiente.
